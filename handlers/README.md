# مشروع مُيَّسر (Moyassar) 🤖📚
### نظام توجيه ذكي لمعلمي المرحلة الابتدائية باستخدام الذكاء الاصطناعي

**مُيَّسر** هو نظام برمجى متطور يعتمد على تقنيات **تنقيب البيانات التعليمية (EDM)** و**تعلم الآلة**، مصمم خصيصاً لدعم معلمي المدارس الابتدائية في (حي الأندلس - طرابلس) من خلال تقديم رؤى تحليلية وتوصيات دقيقة.

---

## 🚀 المميزات الرئيسية (Core Features)
* **تحليل الفجوات (Gap Analysis):** باستخدام خوارزمية `XGBoost` لتقييم مستويات الطلاب.
* **تتبع المشاركة:** استخدام نماذج `LSTM` لتحليل اتجاهات مشاركة الطلاب بمرور الوقت.
* **توليد التغذية الراجعة:** استخدام `DistilBERT` لتقديم ملاحظات نصية ذكية للمعلمين.
* **تقارير تلقائية:** إنشاء ملفات PDF احترافية لتقييم الحصص الدراسية.

## 🛠 الهيكلية البرمجية (System Architecture)
المشروع مصمم بهيكلية **Modular** لتسهيل التطوير والمشاركة:
* `/models`: تحتوي على نماذج تعلم الآلة والأوزان التدريبية.
* `/handlers`: إدارة الأوامر التفاعلية لبوت التيليجرام.
* `/data`: إدارة قواعد البيانات (Excel) والمناهج الدراسية.
* `/utils`: الأدوات المساعدة مثل منشئ الـ PDF ومعالج النصوص.
