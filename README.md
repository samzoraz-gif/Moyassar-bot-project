# 🤖 Moyassar Bot Project | مشروع بوت ميسر الذكي
> نظام متكامل للإرشاد الذكي يعتمد على تحليل البيانات وإدارة العمليات الآلية.

---

## 📁 هيكلة المشروع (Project Structure)
تم تنظيم الكود بشكل موديولات (Modular Layout) لتسهيل الصيانة والتطوير:

* **`handles/`**: المجلد المسؤول عن إدارة العمليات والربط (Orchestration).
* **`models/`**: يحتوي على خوارزميات التحليل (مثل `trend_analyzer.py`).
* **`utils/`**: الأدوات المساعدة لمعالجة البيانات وتصويرها (Visualizer).
* **`data/`**: لإدارة قواعد البيانات والملفات التشغيلية.

### 📄 الملفات الرئيسية في الجذر:
* `main.py`: نقطة انطلاق البوت المشغلة.
* `config.py`: ملف الإعدادات والتهيئة.
* `Master_Database.xlsx` & `School_Data.xlsx`: قواعد البيانات المرجعية للبوت.
* `Amiri-Regular.ttf`: ملف الخط المستخدم في توليد التقارير أو الصور.

---

## 🛠️ تقنيات المشروع
المشروع مبني باستخدام لغة **Python** ويعتمد على الموديولات التالية:
* **تحليل البيانات:** Pandas / Openpyxl.
* **المعالجة:** موديولات مخصصة للتحليل والربط الذكي.
